## Suggested project structure

```text
projects/
└── project-name/
    ├── index.md
    ├── process.md
    └── results.md
```

A project page should briefly describe:

* `objective`;
* `responsible contributors`;
* `current status`;
* `methods`;
* `results`;
* `open tasks`;
* `related documents`.
